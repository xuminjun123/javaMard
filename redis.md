# Redis  入门

## 概述

> Redis 是什么 ？

Redis  ( Remote  Dictionary  Server ) ， 即 远程字典服务！

是一个开源的使用ANSI  C语言 编写，支持网络，可基于内存亦可持久化的日志型  ， Key-Value 数据库 ，并提供多种语言的API ，免费和开源 ，是当下最热门的NoSQL 技术之一 ！

也被称为 结构化数据库。



>  Redis 能干嘛 ？ 

1. 内存存储 ，持久化 ，内存中断电即失 ，所以说持久化很重要
2. 效率该，可以用于告速缓存

3. 发布订阅系统
4. 地图信息分析
5. 计时器，计数器 （浏览量  ）

.....

> 特性

1. 多样化的数据类型

2. 持久化

3. 集群

4. 事务

5.  .....
